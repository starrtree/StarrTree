# StarrTree 3D Assets

Upload the 3D avatar model here as:

```text
assets/AxStarr.glb
```

The homepage is already wired to look for that file. Until it exists, the site falls back to the current AxStarr PNG.

Recommended GLB settings:
- Binary `.glb`
- Embedded textures
- Draco/Meshopt compression if available
- Ideally under 10–20 MB for web performance
